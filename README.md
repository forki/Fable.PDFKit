# Fable.PDFKit

Fable bindings and helpers for [PDFKit](https://pdfkit.org/).

## Installation

```
npm install --save pdfkit svg-to-pdfkit # or
yarn add pdfkit svg-to-pdfkit

paket add Fable.PDFKit --project [yourproject]
```
